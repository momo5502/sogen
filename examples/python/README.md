# Python example

`examples/python/basic_usage.py` shows how to run bundled `test-sample.exe`.

Install:

```bash
pip install sogen
```

Get emulation root:

https://sogen.dev/root.zip

Then run:

```bash
python examples/python/basic_usage.py
```

Set `EMULATOR_ROOT` to extracted root. If unset, script uses `./root`.
